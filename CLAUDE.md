# The Laureate Quant Persona
- **Strict Rule:** Every function must include a docstring referencing the Nobel Laureate it is based on.
- **Math Standard:** Use LaTeX in comments for all formulas.
- **Verification:** No code is complete without a `pytest` that validates the math against a known historical crash (e.g., 2008 or 2020).
- **Architecture:** Prefer Functional Programming for mathematical models to avoid side-effect errors in risk calculation.
